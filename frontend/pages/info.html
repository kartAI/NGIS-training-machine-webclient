Step by step instruction

  1. Enter coordinates into the application, this can be done either by selecting an area on a map, entering coordinates manually or uploading a geojson file
  2. Enter data parameters for how much of the training data you would like to use for training and validation
  3. (WMS Data sources only) Select the layers you would like to be included in your label data
  4. Select the tile size and image resolution you want for the generated files
  5. Download your selected data

What can you use the generated data for?

The generated data can be used for training machine learning models.

About KartAi

KartAi is a research project conducted by Kartverket, Norkart, Kristiansand Municipality, and the University of Agder with financial support from the Regional Research Fund Agder. The main purpose of the project is to make the processing of building cases in the municipality more efficient. This will be achieved by improving the quality of the property register (matrikkelen), and map databases using Ai. The KartAi project has three sub-goals:

1. Buildings will be identified by the help of different data sources like, laserdata, aerial photo and crow sourcing. This will be done by creating a “geodata lake”, with data sources that are optimized for Ai and big data analysis.
2. Provide a better decision-making basis for municipal case management. The project will identify buildings under 50 square meters in a selected area in Kristiansand by using and developing new Ai algorithms from the mentioned “geodata lake” This will typically include buildings that are exempt from the municipal application requirements.
3. Ensure more efficient processing of cases through reliable and automated quality improvement of the national property register (matrikkelen) and map databases in dialog with residents.

This application belongs to the first subgoal, the generated data can be used to train Ai to recognize buildings.
