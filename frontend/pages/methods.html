Info about methods

Label Sources:

  • NGIS - Data directly from the central map database of the Norwegian mapping authority.
  • WMS - Openly released data from Kartverket stored in a Web Map Service
  • FGB - Data stored in FlatGeoBuf format, this data is part of the KartAI dataset.

Orto Sources:

  • WMS - Openly released data from Norkart stored in a Web Map Service
  • COG - Data stored in the Cloud Optimized GeoTiff format, this data is part of the KartAI dataset.
  • Satellite - ...

Metadata file:

You also have the option to upload a metadata file from a previous project. If you do this then the application will preload all the settings from the metadata but you can still edit parameteres such as coordinates and application data settings.

Methods:

  • Draw Shape On Map - Draw your desired coordinates directly on the map.
  • Write Coordinates - Input your coordinates as a text string.
  • Upload geoJSON File - Upload and parse a geoJSON file for your coordinates.
