Settings

?Training percentage determines how much of the data to be allocated to training data. This data is used to train your Ai model
?Validation percentage determines of much of the data to be allocated to validation data. This data is used to validate your Ai models performance during training
?Building percentage ensures that the generated data has the specified percentage of buildings. If you put in "10%", the generated images have to contain atleast 10% buildings
?Only applicable if you chose "WMS" as a data source
?Tile size determines how big the generated tiles are in pixels. An input of 500, outputs an image of 500x500 pixels. Please input a value between 100 - 1104
?Image resolution determines the level of detail present in the genereated images. A resolution of 0.2 means that each pixel represents an area of 2m, therefore lower tile size and higher image resolution results in a more detailed picture. Please input a value between 0.1 - 0.5
